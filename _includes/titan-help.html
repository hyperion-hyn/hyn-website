A simple guide to “cryptographic location share”

User A (Share encrypted location message), User B (View decoded map)
Step 1: Share location
1. User B copy individual public key address and share it with User A via Messaging Apps
2. User A select a location on Titan map, and click “share private location”
3. User A paste User B’s public key address
4. User A fill in notes to share with location (optional)
5. User A Click “encrypt and share” to generate location code and share to B
Step 2: View decoded map
1. User B copy location code received from User A
2. Titan will automatically shows the code. User B click “Decode location in Map”
3. Or User B manually paste the code to the top frame
4. If location shows, your location is decoded successfully.
5. Click “Titan” button to delete data after location share
