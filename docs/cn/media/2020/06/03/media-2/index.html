海伯利安

简介

海伯利安是致力于为全球100亿用户提供完整公民地图数据服务的去中心化地图平台，旨在实现“One Map”愿景。海伯利安通过包含空间信息及价值流转的空间共识协议，重新定义用户创建及使用地图服务的方式，与搭建完全开源及去中心化地图的首要原则完美契合。

首款基于密码学保障的百分百隐私保护地图App。具有手机端操作定位及搜索位置等功能。此外，泰坦提供个人位置不离开手机，及一键“阅后即焚”的特性。通过密码学技术将达到保障用户分享的个人隐私位置不被第三方窃取的目的。

- 安全的公钥地址代替位置接收者身份

- 离线地图保障用户数据不离开手机

- 密码学保障的位置隐私分享

Download

Map3去中心化地图服务网络启用全新的空间数据模型，旨在大规模、高性能及低成本地将基础设施提升1000倍。Map3是由One Map算法技术支撑的去中心化地图/位置服务PaaS, 旨在支持构建Map3去中心化节点的大型网络。

- 可创建使用基于自定义用途和规则的随机地图层

- 可提升1000倍性能的规模化去中心化基础设施

- 跨平台地图技术堆栈间的无缝转接（如Mapbox,Openlayer）

- 强密码学原理支撑的地图数据加密

运行节点 运作原理

Atlas是以空间共识和弹性空间分片为主要特征的高性能地图区块链，可解决空间分片，账户资产转移，资产确认等与空间相关的服务需求。具有抗审查、开放参与、高容错等特点。未来待Atlas链搭建完成，将支持海伯利安成为服务100亿人的去中心化全球地图经济体系。

Atlas将在技术上实现高可伸缩性、低延迟性、低成本、隐私保护、可扩展性和交互性，支持全球体量巨大的地图定位请求和位置搜索请求。未来主网上线之时，用户将能成为Atlas节点，参与提升区块链网络的安全、性能等，并获得HYN奖励。Atlas节点的奖励细则还有待确认，但将与Map3网络的边缘节点以及核心节点关联统一。

地图的演变

去中心化地图的演变历史

  • 地图 1.0

    数字地图

    数字地图具有互动和搜索功能，但缺乏开放、隐私及大规模社区贡献的特性。

  • 地图 2.0

    开源地图

    开源地图开创了社区参与和免费开放的地图数据，但在过去的十年中技术未得到有效改进。其开放的社区贡献模式缺乏对社区进行有效治理和激励的社会及经济架构。

  • 地图3.0

    共识地图

    共识地图具有合理的技术、社区及经济系统架构，由去中心化地图技术、自治社区及可持续的地图经济共同驱动发展。
